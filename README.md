# Tienda-ecommerce_PruebaTecnica_PHP_Laravel
El objetivo de esta prueba es evaluar la habilidad del desarrollador para construir una API RESTful que simule las funcionalidades de la API de Fake Store (https://fakestoreapi.com/docs). La API por desarrollar permitirá crear usuarios, gestionar productos y categorías, y agregar productos a un carrito de compras.
